## 📁 Repository Structure

LeetSync automatically creates a folder for every solved problem.

```text

│
├── 📂 0001-two-sum
│   ├── README.md
│   └── two-sum.java
│
├── 📂 0009-palindrome-number
```

Each problem folder contains:

- 📄 Problem statement (generated by LeetSync)
- 💻 Java solution
- ✅ Accepted submission
